弹出层

信息框

信息： onclick="XUI.alert('文字内容');"

成功： onclick="XUI.succeed('文字内容');"

失败： onclick="XUI.fail('文字内容');"

警告： onclick="XUI.warn('文字内容');"

确认框

确认： onclick="XUI.confirm('确定继续?', callback));"

通知框

通知： onclick="XUI.notic(title, content, time);"

参数说明： title : '', // 标题文字
content : '', // 内容（可以是html片段）
time : 0 // 单位毫秒后自动关闭，默认为0，不会自动关闭。

加载层

加载：

有文字： onclick="XUI.load(msg, time);"

参数说明： msg : '', // 显示文字
time : 0 // 单位毫秒后自动关闭，默认为0，不会自动关闭。一般需要调用XUI.close(index)来关闭，其中index为XUI.load的返回值

弹出页

基本窗口： onclick="Dialog.open(options);"

外部链接： onclick="Dialog.openOuter(options);"

options参数： key : 'default_key', // 若key相同则刷新，不同则继续弹出（必须，openPage方法无）
title : '窗口', // 标题
maxmin : true, // 是否有最大/小化按钮
model : true, // 是否为模态
width : 830, // 弹出层宽
height : 560, // 弹出层高
url : null, // 请求地址（必须）
data : {}, // 请求数据（openPage方法无）
callback : function() {} // 打开页面后回调

弹出层链接

默认： 点击弹出 外部链接 target="dialog"

其他属性： 点击弹出1 点击弹出2 点击弹出3 target="dialog" rel="new_dialog" title="新弹出层" width="460" height="330" model="false"

属性说明： rel属性：对应options.key
model属性：对应options.model
width属性：对应options.width
height属性：对应options.height
title属性：对应options.title
href属性：对应options.url
